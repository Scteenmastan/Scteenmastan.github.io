简介

此模型使用了4个ReLu隐藏层，和1个Softmax输出层搭建神经网络，拥有超过1000个模型参数和大约3000个训练数据，训练数据来源于互联网

用法

首先安装相关库

pip install tensorflow
pip install Pillow
pip install numpy

如果将来运行时出现了No module named 'xxx'的错误，可以尝试安装xxx库

pip install xxx

运行

直接点击start.bat，根据提示选择使用CLI(目前CLI的功能发生变化，请使用GUI)或者GUI。

CLI运行过程

  1. 运行时会先加载数据，这会花费一段时间，等到程序输出Input the path of the sudoku image:时，请输入数独图片的文件路径，然后等待程序运行（可能会花费一段时间），最后会格式化输出数独识别结果
  2. 上一阶段完成后，程序会提示If there is ······ no error, type 'n'，然后提示Command:，如果识别结果的第3行第6列把5识别成了1，请在Command:后输入3 6 5，然后回车。程序会自动把这些错误保存到训练集中
  3. 如果还有错误，请继续按照2的步骤输入错误，直到没有错误为止， 这时请输入n，然后回车，程序会自动保存训练集至磁盘，然后退出该阶段
  4. 程序会提示Do you want to train the model? (y/n)，如果想要训练模型，请输入y，然后回车，程序会自动训练模型，训练完成后会自动退出，否则，请输入n，然后回车，程序会自动退出

GUI功能简介

  1. GUI模式下，请拖动一张图片至表格上方空白处，然后等待一段时间，程序会自动识别图片中的数字并显示在表格中。
  2. 如果程序识别错误（不包括未识别到数字），请选中识别错误的表格单元，点击revise按钮，然后在对话框中输入正确的数字，点击确定即可。
  3. 如果程序未识别到数字，请双击未识别到数字的表格单元，直接输入数字。
  4. 是否点击revise按钮的区别在于，点击revise按钮后，程序会记录错误并用于自我学习，而不点击revise按钮，程序将忽略错误。
  5. 点击solve按钮后，程序自动解数独，解出的数独会显示在表格中。
  6. 如果数独正确解出，程序会自动创建解题报告，点击detail按钮即可查看解题报告。
  7. 菜单栏File中的Open按钮可以打开一个数独图片，效果和拖动图片相同，Save按钮可以将解出的数独保存（即将实现），options中的train按钮可以根据训练数据重新训练模型。
  8. 新增了settings和style两个按钮，你可以自定义程序设置以及界面风格。
  9. 点击Update按钮时，程序将会自动检查是否有新版本的数据集下载，然后自动替换替换磁盘上的训练数据为最新数据。

设置简介

注意：所有设置只对GUI模式有效

  1. auto-train是否为true表示程序是否自己选择合适的训练时机，训练时会自动开始，不会通知用户。
  2. auto-update是否为true表示程序是否自动更新训练数据，程序更新数据时会提示用户。
  3. 请不要更改含有-read-only后缀的选项，他们由程序用来判断合适的执行不同自动化操作的时机。

注意事项

  1. 训练结果里面的val_accuracy会很高（100％），这是正常现象，因为模型的训练集和测试集的字体十分接近，但是识别数独图片成功率可能不是100％。
  2. 训练用的数独图片的字体尽可能相似，否则可能会降低准确率，字体参考https://5b0988e595225.cdn.sohucs.com/images/20171008/4b7e783b0d7f4d8995bef6ab1df8b8a9.jpeg。
  3. 大概每找出50-100个错误就差不多可以训练一次（如果开启自动训练模式则不必考虑此项）。
  4. 在GUI模式下，如果开启自动更新，程序将会自动保持训练集和测试集为最新状态，这意味者原来的训练数据会丢失，但可能提高准确率。
  5. 最新版本数据集2.0.0: 立刻下载!
